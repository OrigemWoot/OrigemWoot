## Key Features

* Autovote
* AFK Autorespond
* Emotes
* Split Chat
* ETA
* User Join
* User Leave
* Booth Alert
* Show Mehs
* DJ Stats
* Music Length Alert
* Private Chat
* Desktop Notifications
* Emotes menu
* More than 80 commands
